# amazon-sales-dashboard(Power BI)
Power BI dashboard analyzing Amazon sales data

## 📊 Project Overview
This project analyzes Amazon sales data using Power BI to generate business insights.

## 🔍 Key Insights
- Total Sales: 32.87M
- Top Region: Asia
- Balanced category performance
- Equal distribution across payment methods

## 🛠 Tools Used
- Power BI
- Data Cleaning
- Data Visualization

## Business Insights
- Asia has the highest slaes contribution.
- Discounts are highest in a Beauty category.
- Sales trend is stable with slight seasonal variation
- Asia region shows a total loss of 0.41M, mainly driven by high discounts across categories.
